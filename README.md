# Personal Finance Tracker

A console-based Personal Finance Tracker application developed in Java.

## Features

- Add income and expense transactions
- Delete transactions
- Calculate current balance
- Generate financial reports
- Add custom categories
- View expenses by category

## Technologies

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Enum
- LocalDate

## Project Structure

- FinanceTracker.java
- Transaction.java
- TransactionType.java
- PersonalFinanceTracker.java

## Example

```text
===== PERSONAL FINANCE TRACKER =====

1. Add Transaction
2. Delete Transaction
3. Show Transactions
4. Show Balance
5. Generate Report
6. Add Category
7. Exit
```
